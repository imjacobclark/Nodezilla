Nodezilla
=========

A rich featured response and load testing suite in Node.js.

### Requirements:
* 1GB+ RAM & 2x CPUs
* Node.js 0.10.*+

### Installation:

```bash
$ wget https://github.com/imjacobclark/Nodezilla/archive/v0.0.2.tar.gz && tar -zxvf v0.0.2.tar.gz && cd Nodezilla-0.0.2
$ npm install
$ node server.js
```
